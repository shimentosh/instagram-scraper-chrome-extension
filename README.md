## What is Instagram Scraper?

**Instagram Scraper** is a free Chrome extension that lets you collect public Instagram data directly from your browser — profiles, posts, followers, following lists, hashtags, comments, and likers — and export it all to CSV, JSON, HTML, or TXT in one click.

No third-party server. No account credentials stored. No API key required. Just install, open Instagram, and start scraping.

---

## 7 Scrape Types — All in One Extension

### Profile Scraper
Extract bio, follower count, following count, post count, verification status, business category, website, user ID and more — for any public Instagram account.

### Posts Scraper
Collect post URLs, captions, hashtags, mentions, like counts, comment counts, view counts, timestamps, media type (image/video/carousel), location, and shortcodes.

### Followers Scraper
Export a full followers list with username, full name, verification status, privacy status, follower/following counts, and profile URLs.

### Following Scraper
Same detailed data as the followers scraper — for any account's following list.

### Hashtag Scraper
Discover top posts under any hashtag with full engagement metrics, captions, timestamps, media types, and owner details.

### Comments Scraper
Pull all comments from any public post — including text, username, timestamp, likes, and nested replies. Sort by popular or most recent.

### Likers Scraper
See who liked any public post. Export up to 500 accounts with full profile details including verification and follower counts.

## Features

- Scrape up to **5,000 items** per session
- Choose exactly **which fields** to collect
- Filter: **verified accounts only**, skip private accounts
- Filter posts by type: **images, videos, or carousels**
- Sort comments by **popular or most recent**
- Include or exclude **comment replies**
- **Real-time progress bar** and activity logs
- **Stop scraping** at any time with one click
- **Auto-detects** the current Instagram page type
- **Light and dark mode**

---

## Privacy & Security

Instagram Scraper runs entirely inside your browser using your existing Instagram session. No data is ever sent to an external server. No account credentials are stored. Works only on public accounts and content you already have access to.

---

## Installation

1. Download or clone this repository
2. Go to `chrome://extensions` in Chrome
3. Enable **Developer Mode** (top right)
4. Click **Load unpacked** and select the project folder
5. Open Instagram and click the extension icon
